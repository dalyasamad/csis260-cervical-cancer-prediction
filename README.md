# CSIS 260 - Cervical Cancer Risk Prediction

## Project Overview
This project was developed for the CSIS 260 Machine Learning course. The objective is to build predictive models using a real-world medical dataset related to cervical cancer risk factors.

The project covers:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Regularization techniques
- Machine learning model training
- Hyperparameter tuning
- PCA feature reduction
- Model evaluation and comparison
- Visualization and interpretation

---

## Dataset
Dataset: Cervical Cancer Risk Factors Dataset

The dataset contains medical and behavioral risk factors associated with cervical cancer diagnosis.

---

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Seaborn

---

## Machine Learning Models
The following models were implemented:
- Logistic Regression
- Random Forest
- Decision Tree
- Support Vector Machine (optional)

---

## Evaluation Metrics
Models were evaluated using:
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

---

## Hyperparameter Tuning
Grid Search and Random Search techniques were used to optimize model performance.

PCA was also tested for dimensionality reduction and feature selection.

---

## Results
The final model achieved strong predictive performance after preprocessing, tuning, and evaluation.
